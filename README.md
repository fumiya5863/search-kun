# search-kun
多種多様なIT全般の情報サイトにアクセスすることが出来る

## 説明
## 環境構成
- Node.js 12.14.1
- npm 7.5.4

## 利用方法
ITの情報を取得するためのサイト紹介。全然サイトの情報が載ってないので皆さんで一緒に情報を掲載していきましょう。electronで使用することが出来ます。

## 実行
```
git clone https://github.com/fumiya5863/search-kun.git
npm install
npm start 
```
